# Projet | appdashboard

Description du projet
ce ci est un dashboard d'enregistrement des employés.
son but est d'enregistrer à l'aide d'un interface dynamique les employés de l'entreprise et
régerer un bulletin PDF qui resume toutes les informations sur un employé.

# comment l'utiliser ?

1. créer un .venv
2. installe les bibliothèques nécessaire avec la commande suivante

- pip install -r requirements.txt

3. Créer un config.env et l'éditer en se referent au fichier config.env.exemple

NB: ce projet utilise la base de données mysql
pensé à ajuster les paramètres dans settings.py et config.env

4. Exécuter le projet
   - python manage.pu runserver

5. rendez vous sur :
   - [text](http://127.0.0.1:8000/)
